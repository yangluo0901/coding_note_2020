#HTML/CSS    

###1. Disable and gray out a link    

```scss
.disable{
    color: currentColor;
    cursor: not-allowed;
    opacity: 0.5;
    text-decoration: none;
}
```


###2. Place an element center when its position is __absolute__    

```css
.pager{
    position: absolute;
    bottom: 100px;
    left:0;
    right:0;
    margin: auto;
}
```
